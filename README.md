# treatment

・Shiftキーを押し、スニーク状態になると花火が飛ぶ仕組みとなっております。<br>
・連打すると花火が連発して飛びます。<br>
・屋根があるところなどプレイヤーと距離が近くなるとダメージを受けるので注意してください。<br>
※Macのみ動作確認済みです。
